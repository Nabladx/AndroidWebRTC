# Video chat backend

## Description:
**Implementation of an Omegle-like video chat backend using WebRTC standard and Clean Architecture principles.**

## Tech:
- JavaScript
- RxJS
- React.js
- Redux (Redux Toolkit)
- WebSocket
- WebRTC
