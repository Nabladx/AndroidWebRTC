# Video chat Android client
WebRTC random video chat application client implementation using Clean Architecture and functional reactive programming principles.

## Tech:

- Kotlin
- Coroutines (Flow)
- Arrow-KT
- Hilt
- WebSocket (OkHttp)
- WebRTC
- Android
- Jetpack Compose
- Accompanist
